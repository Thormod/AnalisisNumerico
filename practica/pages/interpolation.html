Newton Divided Differences

Escribir las coordenadas de los puntos y el valor con el que se evalua, pueden agregarse o disminuirse la cantidad de puntos oprimiendo add point o drop point respectivamente,y luego oprime "Calculate". Genera un polinomio de Newton que se ajusta a los puntos dados.

LaGrange Polynomial

Escribir las coordenadas de los puntos y el valor con el que se evalua, pueden agregarse o disminuirse la cantidad de puntos oprimiendo add point o drop point respectivamente,y luego oprime "Calculate" Genera un polinomio de LaGrange correspondiente a los puntos dados

Neville Interpolation

Escribir las coordenadas de los puntos y el valor con el que se evalua, pueden agregarse o disminuirse la cantidad de puntos oprimiendo add point o drop point respectivamente,y luego oprime "Calculate" Genera un polinomio de Neville con los puntos dados.
